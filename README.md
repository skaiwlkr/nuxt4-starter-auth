# Nuxt 4 Starter Auth

## Overview

Nuxt 4 Starter Auth is a template for a Nuxt 4 application with authentication.

## Features

- Authentication
- Authorization
- Login
- Register
- Forgot Password
- Reset Password
- Logout
- Two-Factor Authentication (Coming soon)

## Tech Stack

- Frontend: Nuxt 4
- Database: PostgreSQL
- UI Framework: Tailwind CSS
